Feb 25, 2016

类中类

package com.pengfeixing;

import java.util.HashMap;
import java.util.Map;
import java.util.ArrayList;

public class DirectedGraph {
public static class Vertex{
private int value;
public Vertex(int val){
this.value = val;
}
public int getValue(){
return this.value;
}
}

public static void main(String[] args){
    DirectedGraph test = new DirectedGraph();
    test.addVertex(new Vertex(0));
    test.addVertex(new Vertex(1));
}

}

为何Vertex要定义成静态类？

答：这里可以理解为类方法，假设类A含有内部类B，则将B声明为静态类，即可将B看成A的类方法，生成类B只需要用类A名就可以了，而不需要生成一个类A，即A.B b = new A.B();相反，如果B不是静态类，生成类B则需要先生成类A，即A.B b = new A().new B()。如果该内部类定义为私有，则只能在外部类内访问。
