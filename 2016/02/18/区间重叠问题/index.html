区间重叠问题

  • Algorithm

问题描述

给定n个区间，找出重叠数量最多所对应的点。形象化描述是有4个飞机的schedule:[1,2], [3,4], [2,5], [3,6]，空中最多有多少架飞机。这里答案是3，因为在时刻3和4，空中飞机的数量最多，为3。为了简单起见，有两个假设：

假设1: 区间起点和终点都是整数；

假设2:假设同一时刻有一个飞机降落，另外一架飞机起飞，则先降落后起飞。

暴力解法

首先想到的是greedy algorithm，进行三层遍历循环。先将所有区间按照起始数值排序，然后依次遍历区间数组，在每次遍历某个区间时，又遍历该区间的每个数值，将该数值和区间数组中的剩下的区间进行比较，看是否重合。该算法在OJ上能通过87%的case，但是当测试集变大之后就会超时。算法实现如下：

/**
 * Definition of Interval:
 * public classs Interval {
 *     int start, end;
 *     Interval(int start, int end) {
 *         this.start = start;
 *         this.end = end;
 *     }
 */

class Solution {
    /**
     * @param intervals: An interval array
     * @return: Count of airplanes are in the sky.
     */
    public int countOfAirplanes(List<Interval> airplanes){
        if(airplanes.size() == 0) return 0;
        sortList(airplanes);
        int maxCount = 1;
        while (airplanes.size() > 1){
            Interval head = airplanes.get(0);
            for (int i = head.start; i < head.end; i++){
                int count = 1;
                for (int j = 1; j < airplanes.size(); j++){
                    Interval time = airplanes.get(j);
                    if (time.start <= i && time.end > i){
                        count++;
                    } else if (time.end <= i){
                        airplanes.remove(j);
                    }
                }
                maxCount = Math.max(count, maxCount);
            }
            airplanes.remove(0);
        }
        return maxCount;
    }
    //sort a Interval list by the start point(ascending order)
    private void sortList(List<Interval> list){
        for (int i = 0; i < list.size() - 1; i++){
            for (int j = i + 1; j < list.size(); j++){
                if (list.get(i).start > list.get(j).start){
                    Interval tmp = list.get(i);
                    list.set(i, list.get(j));
                    list.set(j, tmp);
                }
            }
        }
    }
    /*
    TEST CASE:
        [1,1]
        [1,2],[2,3]
    */
}

算法优化

为了对算法进行进一步的优化，我们可以对问题进行类比，区间重叠问题可以类比为括号最大深度的问题，即求解“()((()))()”中最大的括号深度，答案是3。算法实现如下：

  • Definition of Interval:
  • public classs Interval {
  • int start, end;

    class Pair {
    int time;
    int isStart;
    Pair(int time, int isStart) {
    this.time = time;
    this.isStart = isStart;
    }
    public static Comparator<Pair> PairComparator = new Comparator<Pair>(){
    public int compare(Pair p1, Pair p2){
    if (p1.time == p2.time) return p1.isStart - p2.isStart;
else
    return p1.time - p2.time;
    }
    };
    }
    class Solution {
    /**
    • @param intervals: An interval array
    • @return: Count of airplanes are in the sky.
      /
      public int countOfAirplanes(List<Interval> airplanes){
      List<Pair> list = new ArrayList<>(2       airplanes.size());
      for (Interval i : airplanes){
      list.add(new Pair(i.start, 1));
      list.add(new Pair(i.end, 0));
      }
      Collections.sort(list, Pair.PairComparator);
      int count = 0;
      int ans = 0;
      for (Pair p : list){
      if (p.isStart == 1) {
      count++;
      } else {
      count--;
      }
      ans = Math.max(ans, count);
      }
      return ans;
      }
