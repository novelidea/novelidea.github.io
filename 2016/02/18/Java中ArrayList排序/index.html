Java中ArrayList排序

Java

基础

在Java中，如果需要将一个arrayList排序，我们需要利用Collections类。首先先游离一下主题，先简要介绍一下Collections，Collections不同于Collection，后者（java.util.Collection）是一个集合接口，它提供的是对集合对象进行基本操作的通用接口方法，而前者（java.util.Collections）是一个包装类，它包含有各种有关集合操作的静态多态方法，此类不能被实例化，就像一个工具类，服务于Java的Collection框架。

在本文中，我们讨论ArrayList排序，用到的就是Collections的sort方法，下面的例子简单实现了对ArrayList<Integer> 和ArrayList<String>的升序排序和降序排序。

package com.pengfeixing;
import java.util.*;

public class TEST {
    public static void main(String[] args) {
        ArrayList<Integer> intList = new ArrayList<Integer>();
        intList.add(2);
        intList.add(1);
        intList.add(3);
        System.out.println("Before Soring");
        for (int i : intList) {
            System.out.println(i);
        }
        Collections.sort(intList);
        System.out.println("After Sorting");
        for (int i : intList) {
            System.out.println(i);
        }
        Collections.sort(intList, Collections.reverseOrder());
        System.out.println("Reverse Sorting");
        for (int i : intList) {
            System.out.println(i);
        }

        ArrayList<String> stringList = new ArrayList<String>();
        stringList.add("ABC");
        stringList.add("ABE");
        stringList.add("AAB");
        System.out.println("Before Soring");
        for (String s : stringList) {
            System.out.println(s);
        }
        Collections.sort(stringList);
        System.out.println("After Sorting");
        for (String s : stringList) {
            System.out.println(s);
        }
        Collections.sort(stringList, Collections.reverseOrder());
        System.out.println("Reverse Sorting");
        for (String s : stringList) {
            System.out.println(s);
        }

    }
}

进阶

下面我们主要讨论如何对对象的某个属性进行排序，我们不能直接将对象实例名字作为传入sort函数，这样程序会由于sort函数没有排序比较的依据而报错。我们可以有两种选择，如果只需要根据单一实例变量进行排序，我们只需要让需要比较的类继承Comparable，然后在该类中重写Comparable的comparaTo方法，例子如下：

package com.pengfeixing;

import java.util.Comparator;

/**
 * Created by novelidea on 2/18/16.
 */
public class Student implements Comparable {
    private String name;
    private int age;
    public Student(String name, int age){
        this.name = name;
        this.age = age;
    }
    int getAge(){
        return this.age;
    }
    void setAge(int age){
        this.age = age;
    }

    @Override
    public int compareTo(Student comparestu){
        int compareAge = ((Student)comparestu).getAge();
        return this.age - compareAge;
    }
}

package com.pengfeixing;

import java.util.*;

public class TEST {
    public static void main(String[] args) {
        ArrayList<Student> studentList = new ArrayList<Student>();
        studentList.add(new Student("Tom", 12));
        studentList.add(new Student(("Jim", 10));
        studentList.add(new Student("Sam", 14));

        Collections.sort(studentList);

        for(Student stu : studentList){
            System.out.println(stu);
        }
    }
}
如果我们排序的依据是类的多个属性，则需要重写Comparator中的compare方法。例子如下： 
package com.pengfeixing;

import java.util.Comparator;

/**
 * Created by novelidea on 2/18/16.
 */
public class Student {
    private String name;
    private int age;
    public Student(String name, int age){
        this.name = name;
        this.age = age;
    }
    int getAge(){
        return this.age;
    }
    void setAge(int age){
        this.age = age;
    }

    String getName(){
        return this.name;
    }
    void setName(String name){
        this.name = name;
    }

    /*Comparator for sorting the list by Student Name*/
    public static Comparator<Student> StuNameComparator = new Comparator<Student>() {

        public int compare(Student s1, Student s2) {
            String StudentName1 = s1.getName().toUpperCase();
            String StudentName2 = s2.getName().toUpperCase();

            //ascending order
            return StudentName1.compareTo(StudentName2);

            //descending order
            //return StudentName2.compareTo(StudentName1);
        }};

    /*Comparator for sorting the list by age*/
    public static Comparator<Student> StuAgeComparator = new Comparator<Student>() {

        public int compare(Student s1, Student s2) {
            int age1 = s1.getAge();
            int age2 = s2.getAge();

            /*For ascending order*/
            return age1-age2;

            /*For descending order*/
            //rollno2-rollno1;
        }};

}

package com.pengfeixing;

import java.util.*;

public class TEST {
    public static void main(String[] args) {
        ArrayList<Student> studentList = new ArrayList<Student>();
        studentList.add(new Student("Tom", 12));
        studentList.add(new Student(("Jim", 10));
        studentList.add(new Student("Sam", 14));

        Collections.sort(studentList, Student.StuNameComparator);
        for(Student stu : studentList){
            System.out.println(stu);
        }

        Collections.sort(studentList, Student.StuAgeComparator);
        for(Student stu : studentList){
            System.out.println(stu);
        }
    }
}
