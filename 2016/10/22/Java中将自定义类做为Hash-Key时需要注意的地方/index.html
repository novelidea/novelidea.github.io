Java中将自定义类做为Hash Key时需要注意的地方

  • Algorithm

Java在处理自定义类做为Hash Key时（或者拓宽到任意的对象），比较往往是根据reference比较的，而不是根据value，如果有需要根据value进行比较的需求的话，则需要在类中重写 equals 和 hashCode 两个方法，具体以重写二维矩阵中每个横纵坐标对来举个🌰。
